## Project Rules

### Code Standards
This project follows **Ultracite** code standards (see `.claude/CLAUDE.md`):
- Use explicit types for clarity
- Prefer `const` by default, `let` only when reassigning
- Use async/await over promise chains
- Use function components over class components
- Remove `console.log`, `debugger` from production code
- Add `rel="noopener"` for `target="_blank"` links
